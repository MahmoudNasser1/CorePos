# Error Copy Dictionary — CorePOS (Arabic UI)

> هدف الملف: توحيد صياغة رسائل الأخطاء/الحالات في الواجهة (بدون إنجليزي) لضمان الاتساق وتقليل “اختراع نصوص” في كل صفحة.

---

## 1) Loading / Success
- **Loading**: `جاري التحميل…`
- **Saving**: `جاري الحفظ…`
- **Deleting**: `جاري الحذف…`
- **Submitting**: `جاري الإرسال…`
- **Success (generic)**: `تم الحفظ بنجاح.`
- **Deleted (generic)**: `تم الحذف بنجاح.`

---

## 2) Empty states
- **Empty (no data yet)**: `لا توجد بيانات بعد.`
- **Empty (no results)**: `لا توجد نتائج مطابقة لبحثك.`

---

## 3) Network / Server
- **Network error**: `تعذّر الاتصال بالخادم. تحقق من الإنترنت ثم أعد المحاولة.`
- **Server error (generic)**: `حدث خطأ غير متوقع. حاول مرة أخرى.`

---

## 4) Auth / Permissions
- **Unauthorized**: `انتهت الجلسة أو لا تملك صلاحية الوصول. سجّل الدخول مرة أخرى.`
- **Forbidden**: `لا تملك صلاحية لتنفيذ هذا الإجراء.`

---

## 5) Validation / Not found / Conflict
- **Validation error**: `تحقق من الحقول ثم أعد المحاولة.`
- **Not found**: `العنصر المطلوب غير موجود.`
- **Conflict**: `لا يمكن إتمام العملية بسبب تعارض. حدّث الصفحة ثم أعد المحاولة.`

---

## 6) Ops / Dangerous actions (Admin)
- **Reason required**: `اكتب سبب تنفيذ هذا الإجراء.`
- **Impersonation banner**: `أنت تعمل الآن كـ حساب مستخدم آخر.`
- **Impersonation end**: `تم إنهاء الانتحال والعودة إلى حسابك.`
- **Data repair started**: `تم بدء عملية الإصلاح. يمكنك متابعة حالتها من سجل العمليات.`
