## 1. 📋 موجز المشروع

| البند | التفاصيل |
|-------|---------|
| **اسم المشروع** | Pos-Sahl |
| **الشعار** | *"بيع أسرع، اعرف أكثر"* |
| **النوع** | Web App (يعمل محلياً + سحابي) |
| **التقنية** | Next.js 15 App Router + NestJS + PostgreSQL |
| **اللغة** | عربية 100% (RTL) |
| **نموذج العمل** | اشتراك شهري / سنوي / شراء كامل |
| **مدة MVP** | 8-10 أسابيع |

### الصورة الكاملة
برنامج بسيط وسريع لإدارة **نقاط البيع + المبيعات + المخازن** للمحلات الصغيرة والمتوسطة في السوق العربي. يدعم الباركود، الشاشة اللمسية، الفواتير العربية، حساب الأرباح الدقيق، والتقارير الفورية.

### الجمهور المستهدف
| الفئة | أمثلة | الأولوية |
|-------|-------|---------|
| محلات الإلكترونيات | موبايلات، كمبيوتر، إكسسوارات | 🔴 عالية |
| قطع الغيار | سيارات، كمبيوتر | 🔴 عالية |
| الأدوات والمستلزمات | منزلية، صحية، مكتبية | 🟠 متوسطة |
| محلات الملابس | ملابس، إكسسوارات | 🟠 متوسطة |
| سوبرماركت صغير | بقالة، مواد غذائية | 🟡 لاحقاً |
| صيدليات | أدوية، مستلزمات طبية | 🟡 لاحقاً |

## 3. 👤 User Stories

### الكاشير
```
US-01: أريد البحث عن الصنف بالباركود أو الاسم لأكمل البيع في < 5 ثواني
US-02: أريد طرق دفع متعددة في نفس الفاتورة (كاش + بطاقة)
US-03: أريد طباعة الفاتورة تلقائياً بعد الدفع دون خطوات إضافية
US-04: أريد مراجعة رصيد الخزينة مقارنةً بالواقع في نهاية اليوم
```

### مدير المحل
```
US-05: أريد رؤية قيمة المخزون الحالي في أي لحظة
US-06: أريد تنبيهاً فورياً عند اقتراب أي صنف من النفاد
US-07: أريد تقرير أرباح كل فاتورة وكل صنف بالتفصيل
US-08: أريد إدارة صلاحيات كل موظف بدقة
```

### صاحب المحل
```
US-09: أريد تقرير يومي واضح بالأرباح والمبيعات من موبايلي
US-10: أريد مقارنة مبيعات هذا الشهر بالشهر السابق
US-11: أريد نسخة احتياطية تلقائية يومية
US-12: أريد إضافة فروع جديدة ومتابعة كل فرع بشكل مستقل
```

## 6. 🗺️ خارطة الطريق

### المرحلة 1 — MVP (أسبوع 1-10)
```
أسبوع 1-2:  إعداد + Schema + Auth + Layout عربي + إدارة أصناف
أسبوع 3-4:  شاشة POS كاملة + طباعة + ESC/POS + Barcode
أسبوع 5-6:  مبيعات + مشتريات + خزينة + مرتجعات
أسبوع 7-8:  8 تقارير أساسية + Dashboard + Excel export
أسبوع 9-10: اختبار + أداء + نشر Vercel
```

### المرحلة 2 (4-6 أسابيع)
جرد، تحويلات، تقسيط، شيكات، مندوبون، شيفت، PWA offline.

### المرحلة 3 (مستقبل)
موبايل، ZATCA، SMS، AI مبيعات.
