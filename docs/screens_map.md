# 🖥️ Pos-Sahl — خريطة الشاشات والتنقل

> **التقنية:** Next.js 15 App Router | **التاريخ:** 17 أبريل 2026

---

## هيكل التنقل العام

```
/
├── /login                          ← تسجيل الدخول
├── /setup                          ← إعداد أول مرة (Onboarding)
│
└── /dashboard                      ← التطبيق الرئيسي (محمي بتسجيل الدخول)
    ├── /                           ← الرئيسية / Dashboard
    ├── /pos                        ← شاشة البيع السريع (POS)
    │
    ├── /inventory/                 ← المخزون
    │   ├── /products               ← قائمة الأصناف
    │   ├── /products/new           ← إضافة صنف جديد
    │   ├── /products/[id]          ← تفاصيل الصنف
    │   ├── /products/[id]/edit     ← تعديل الصنف
    │   ├── /categories             ← الفئات والتصنيفات
    │   ├── /stock-count            ← الجرد (P1)
    │   └── /warehouse-transfer     ← تحويل بين المخازن (P1)
    │
    ├── /sales/                     ← المبيعات
    │   ├── /invoices               ← قائمة فواتير المبيعات
    │   ├── /invoices/new           ← فاتورة مبيعات جديدة
    │   ├── /invoices/[id]          ← عرض الفاتورة
    │   ├── /invoices/[id]/print    ← طباعة الفاتورة
    │   ├── /quotations             ← عروض الأسعار
    │   ├── /quotations/new         ← عرض سعر جديد
    │   ├── /quotations/[id]        ← عرض تفاصيل عرض السعر
    │   └── /returns                ← مرتجعات المبيعات
    │
    ├── /purchases/                 ← المشتريات
    │   ├── /invoices               ← فواتير المشتريات
    │   ├── /invoices/new           ← فاتورة شراء جديدة
    │   ├── /invoices/[id]          ← عرض فاتورة شراء
    │   ├── /orders                 ← أوامر الشراء PO (P1)
    │   └── /returns                ← مرتجعات المشتريات
    │
    ├── /customers/                 ← العملاء
    │   ├── /                       ← قائمة العملاء
    │   ├── /new                    ← إضافة عميل جديد
    │   ├── /[id]                   ← تفاصيل العميل + كشف حساب
    │   └── /[id]/statement         ← كشف حساب مفصل
    │
    ├── /suppliers/                 ← الموردون
    │   ├── /                       ← قائمة الموردين
    │   ├── /new                    ← إضافة مورد جديد
    │   └── /[id]                   ← تفاصيل المورد + كشف حساب
    │
    ├── /finance/                   ← الحسابات والخزينة
    │   ├── /treasury               ← رصيد الخزينة + حركاتها
    │   ├── /receipts               ← سندات القبض (استلام من عملاء)
    │   ├── /receipts/new           ← سند قبض جديد
    │   ├── /payments               ← سندات الصرف (دفع لموردين)
    │   ├── /payments/new           ← سند صرف جديد
    │   ├── /expenses               ← المصروفات
    │   └── /expenses/new           ← مصروف جديد
    │
    ├── /reports/                   ← التقارير
    │   ├── /daily                  ← الحركة اليومية
    │   ├── /sales-analysis         ← تحليل المبيعات
    │   ├── /profits                ← تقرير الأرباح
    │   ├── /stock                  ← بضاعة المخزن
    │   ├── /product-movement       ← حركة الصنف
    │   ├── /customer-balances      ← أرصدة العملاء
    │   ├── /supplier-balances      ← أرصدة الموردين
    │   └── /treasury-movement      ← حركة الخزينة
    │
    └── /settings/                  ← الإعدادات
        ├── /company                ← بيانات الشركة
        ├── /branches               ← الفروع
        ├── /warehouses             ← المخازن
        ├── /users                  ← المستخدمون
        ├── /payment-methods        ← طرق الدفع
        ├── /taxes                  ← الضرائب
        ├── /print-settings         ← إعدادات الطباعة
        └── /backup                 ← النسخ الاحتياطية
```
