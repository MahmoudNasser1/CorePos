## Findings from getsahl.com

### General Overview
- **Name**: Sahl (سهل)
- **Nature**: POS, Inventory, and Accounting system for small to medium businesses.
- **Language**: Fully Arabic.

### Key Modules (Detailed)
1.  **Inventory/Warehouse**: 
    - Opening balances for items.
    - Warehouse transfers.
    - Inventory count (الجرد).
    - Serial number tracking.
    - Mobile app for inventory counting.
2.  **Sales/POS**: 
    - Sales invoices.
    - Quotations (عرض أسعار).
    - Sales returns.
    - Sales analysis reports.
    - Item booking (حجز بضاعة).
3.  **Purchases**: 
    - Purchase invoices.
    - Purchase returns.
    - Purchase analysis reports.
4.  **Accounts**: 
    - Customers and Suppliers management.
    - Opening balances for accounts.
    - Payments and Expenses.
    - Receipts and Revenues.
    - Sales Reps (مندوبين).
    - Cash/Safe management (الخزينة).
5.  **Technical Features**:
    - Multi-branch support via internet.
    - Barcode printing and support.
    - Custom invoice design using Microsoft Excel.
    - Automated daily backups.
    - ZATCA (Saudi Arabia) e-invoice compliance (Phase 1 & 2).
    - Desktop software with mobile capability for inventory.

### Pricing (Egypt Version)
- **Monthly Subscription**: 100 EGP / license.
- **Annual Subscription**: 1000 EGP / license.
- **One-time Purchase**: 3500 EGP / license (2 years free support/updates).
- *Inclusions*: All plans include technical support and updates.

### UI/UX Description
- **Layout**: Simple dashboard with colored tiles (tiles for Sale, Purchase, etc.).
- **Style**: Windows-like desktop application look. Large, easy-to-click buttons.
- **Language**: Integrated Arabic support throughout the interface.
- **Customization**: High flexibility in report and invoice layouts via Excel.

### Target Audience & Use Cases
- Retail shops (spare parts, electrical, toys, mobile, sanitary ware, etc.).
- Small and medium-sized warehouses.
- Businesses needing ZATCA compliance in Saudi Arabia.
- Multi-branch retail operations.

### Technical Details
- Offline-first desktop application with internet sync for multi-branch.
- Mobile companion for specific inventory tasks.
- Highly integrated with Microsoft Excel for reporting and document design.
- Minimal system requirements promised.
